TOCBLOCK - LDM Documentation

Overview

The TOCBLOCK is 512 bytes long

Layout

Offset Size Description
0x00 8 "TOCBLOCK" Magic number
0x08 4 Sequence number
0x0C 4 Zeros
0x10 4 Sequence Number
0x14 16 Zeros
0x24 10 Bitmap Name
0x2E 8 Bitmap Start
0x36 8 Bitmap Size
0x48 8 Bitmap Flags?
0x52 10 Bitmap Name
0x5C 8 Bitmap Start
0x64 8 Bitmap Size
0x6C 8 Bitmap Flags?

At the moment there are two bitmaps, config and log.
